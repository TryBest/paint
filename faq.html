Frenquently Asked Questions

How to add curved texts?

You can refer to Curved texts to learn how to add curved texts.

Does this app support layers? How to create or add layers?

This app supports layers. In fact, we call it objects in this app. When you add a new shape/object/path, some image or some texts, they will be added as a new object. In other words, you can think of the objects as layers. And layers are created and added automatically when you draw, sketch and edit your image on the canvas. You don't have to create and add new layers manually.

How to open the other pages of a pdf file instead of the first page?

This app is an image editor. When you open a pdf file with it, it will only open the first page and then convert it to an image. So you can't open the other pages of a pdf file directly. If you really need this, please open the pdf file with another app. For example the Preview app which is a default app on macOS. Then you can choose the page you want and then choose the Edit > Copy menu in that app to copy that page. After that, you can open our app and choose the File > New from Clipboard menu to open the copied pdf page.

How to add and use a custom gradient?

Please refer to Choose a gradients to learn how to add and use a custom gradient.

What's the difference between the free version and the full-featured version?

The free version has some limitations. For example:

  • The free version can't open very big images.
  • The free version will add watermarks onto your images when you save them.
  • The free version can't use some features, including curved texts, shadow, reflection, gradient, image effects/filters and so on.

The full-featured version has no such limitations. It will remove all limitations and you can enjoy all features. ï¿¼

Why no new app is installed after purchasing the full-featured version?

When your purchase the full-featured version in this app, the app will remove all limitations, enable all features and become the full-featured version automatically. It won't install another app. You can continue to use this app with all features.

I purchased the full-featured version. Why many menu commands and features are still greyed out and disabled?

Some menu commands and features are specific to the selected objects or image part on the canvas. So they are only available when some objects or some part of an image are selected. You can select some objects or some part of an image on the canvas and then those menu commands will be enabled.

I purchased the full-featured version. But the app is still the free version. How to restore my purchase?

Maybe your purchase receipt is lost on your current Mac. Please delete this app from your Mac and then re-install it from App Store. Your purchase will be restored automatically.
It this still won't work. Please choose the Help > Feedback Issues... menu in the top menu bar to create a report and then send it to us by email support@trybestapps.com.

Tips

You should use the same Apple ID with which you purchased the full-featured version when re-installing this app, or your purchase won't be restored.

How to apply effects and filters to some part of an image?

  1. Step 1: Please choose a selection tool to select an image part which you want on the canvas.
  2. Step 2: Choose an effect or a filter.
  • You can choose the Object > Effect menu in the top menu bar to choose an effect or a filter.
  • You can also right click the selected image part to show the context menu and then choose the Effect menu to choose an effect or a filter.
