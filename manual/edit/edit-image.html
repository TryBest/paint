Edit image

You can edit the whole image or just a part of it. To do this, you can use commands in the Image and Object menu in the top menu bar. The following operations are supported.

Delete

After chooing some part of the image on the canvas with a selection tool, you can click the delete key on your keyboard to delete the selected image part. You can also choose the Edit > Delete or Cut menu command in the top menu bar to delete it.

Copy

After chooing some part of the image on the canvas with a selection tool, you can choose the Edit > Copy menu command in the top menu bar to copy it.

Crop

You can choose the Image > Crop menu in the top menu bar to start cropping the whole image. It has the same functionality with the crop tool. You can select and adjust the area you want to crop to and then double-click or press Enter to apply the crop.

Resize

You can choose the Image > Adjust Size... menu in the top menu bar to resize the whole image. The resize panel will be opened as below.

In this panel, you can:

  • Input a new size, including width and height. The unit can be pixel, cm, mm, feet, inch and percent.

  • Input a new resolution.

  • Turn on/off the Scale Proportionally checkbox.
    Turning on this checkbox can keep the aspect ratio of the image. In other words, when you input a new width/height, the new height/width will be adjusted accordingly. So that the width to height ratio will be kept. For example, the width to height ratio of the image is 16:9, then it will keep the aspect ratio when you input a new width/height. If you don't want to keep the aspect ratio, you can turn off this checkbox.

  • Turn on/off the Merge All Objects checkbox.
    When the Scale Proportionally checkbox is turn on and the Resample Image checkbox is turn off, this checkbox is enabled. You can trun it on to merge all the objects on the canvas into one object when resizing. You can also turn it off to keep all layers and objects. You are advised to turn off this checkbox so that you can re-edit all objects after resizing.

  • Turn on/off the Resample Image checkbox.
    When this is turned off, only the canvas will be resized. The image won't be resized and will be placed on the new canvas. And you can set its position on the new canvas by clicking the image position button below this checkbox.
    When this is turned on, both the canvas and the image will be resized to the new size.

Adjust Opacity

You can choose the Image > Adjust Image Opacity... menu in the top menu bar to rotate the whole image.

If you want to adjust the opacity for some part of the image instead of the whole image, you can choose a selection tool and use it to select the image part you want on the canvas. Then you can choose the Object > Adjust Opacity... menu in the top menu bar to adjust the opacity of the selected image part.

Rotate

You can choose the Image > Rotate Canvas Left or Rotate Canvas Right menu in the top menu bar to rotate the whole image.

If you want to rotate some part of the image instead of the whole image, you can choose a selection tool and use it to select the image part you want on the canvas. Then you can choose the Object > Rotate & Resize... menu in the top menu bar to rotate the selected image part.

Flip

You can choose the Image > Flip Canvas Horizontally or Flip Canvas Vertically menu in the top menu bar to flip the whole image.

If you want to flip some part of the image instead of the whole image, you can choose a selection tool and use it to select the image part you want on the canvas. Then you can choose the Object > Flip Horizontally or Flip Vertically menu in the top menu bar to flip the selected image part.

Adjust colors

You can choose the Image > Adjust Colors... menu in the top menu bar to adjust the colors of the whole image.

If you want to adjust colors for some part of the image instead of the whole image, you can choose a selection tool and use it to select the image part you want on the canvas. Then you can choose the Object > Apply Effect > Adjust Colors menu in the top menu bar to adjust the colors of the selected image part.

Invert colors

You can choose the Image > Invert Colors menu in the top menu bar to invert the colors of the whole image.

If you want to invert the colors for some part of the image instead of the whole image, you can choose a selection tool and use it to select the image part you want on the canvas. Then you can choose the Object > Apply Effect > Invert Colors menu in the top menu bar to invert the colors of the selected image part.

Apply effect

If you want to apply some effect to the whole image, you can choose the Edit > Select All menu in the top menu bar to select all the image, then choose the Object > Apply Effect menu in the top menu bar to choose and apply an effect to the selected image part.

If you want to apply an effect to some part of the image instead of the whole image, you can choose a selection tool and use it to select the image part you want on the canvas. Then you can choose the Object > Apply Effect menu in the top menu bar to choose and apply an effect.

Remove image background

You can choose the Image > Remove Background menu in the top menu bar to remove the background of the whole image. The app will detect the background of this image with AI. If it finds a suitable background, it will remove the background automatically. If it doesn't find a suitable background, it will show you a tip. At this time.

If the Remove Background menu is not available for you or it doesn't find a suitable background, you can also try to use the magic selection tool to select the background manually and then delete it by clicking the delete key on your keyboard.

Important

This feature is only available on macOS 14 (Sonoma) or later.

Tips

This feature is aimed to remove the background of the whole image, it is different with the Object > Remove Background menu which is aimed to remove the background of some image layer/object. Besides, if there isn't a background image, this feature is also not available.
