Modify attributes

You can modify the attributes of the tools or selected layers/objects in the Attributes panel which is located below the top bar and above the canvas. The available attributes are different for different tools and objects. The most commonly used attributes are the following.

Position attributes

You can select some objects and then drag them on the canvas to adjust their position. Sometimes you want to do this more precisely. Then you can click the left right up down arrow keys on your keyboard to move them.

Line/Border attributes

As shown in the screenshot above, you can modify the line/border attributes, including the following:

  • Line dashed style: The dashes style of the line/border.

  • Line switch: Enable or disable the line/border.
    If this switch is turned off, the other line attributes will be disabled.

  • Line draw type: Choose to draw the line with a color or gradient from the poped-up menu.

  • Line color/gradient: The color/gradient of the line/border.
    To get more details, you can refer to Choose a color and Chose and use a gradient.

  • Line width: The width of the line/border. The unit is pixel.

  • Line arrow style: The arrow style of the line/border.

  • Line cap: The cap style of the line.
    It specifies the shape of the endpoints of an open path when stroked. To know more about this, please refer to line cap style.

Outline attributes

As shown in the screenshot above, you can modify the outline attributes for a line, curve, broken line and path. The outline attributes include the following:

  • Outline switch: Enable or disable the outline.
    If this switch is turned off, the other outline attributes will be disabled.

  • Outline width: The width of the outline. The unit is pixel.

  • Outline color: The color of the outline.
    To get more details, you can refer to Choose a color.

Tips

When the outline attribute is enabled, if you set the line color to the clear color, you will get a hollow line.

Fill attributes

You can modify the fill attributes, including the following:

  • Fill switch: Enable or disable the fill.
    If this switch is turned off, the other fill attributes will be disabled.

  • Fill type: Choose to fill with a color or gradient from the poped-up menu.

  • Fill color/gradient: The color/gradient of the fill.
    To get more details, you can refer to Choose a color and Chose and use a gradient.

Smooth attribute

This attribute is turned on by default. It is used to smooth the line, border and texts. You are advised to keep this attribute on. When this attribute is turned on, the line, border and texts will be rendered with anti-aliasing.

Corner attribute

This attribute is only available for the round rectangle. The corner radius is the radius of the four corners. Every corner have two radius values:

  • X: The horizontal corner radius.
  • Y: The vertical corner radius.
    The unit is pixel.

Text attributes

As shown in the screenshot above, you can modify the text attributes, including the following:

  • Font name: The text content.

  • Font style: The font style of the text, such as bold, italic, etc.

  • Font size: The font size of the text. The unit is pixel.

  • Text draw type: Choose to draw the text with a color or gradient from the poped-up menu.

  • Text color/gradient: The color/gradient of the text.
    To get more details, you can refer to Choose a color and Chose and use a gradient.

  • Text alignment: The alignment of the text, including left, center, right and justify.

  • Line and paragraph spacing: The spacing of the line and paragraph.

  • Text outline: The outline attributes of the text, including the following:

    • Outline switch: Enable or disable the text outline.
    • Outline details: Click to open the text outline attributes panel as shown in the following screenshot.

    • Outline color: The color of the text outline.

  • Auto resize: Use fixed size for the text box or auto resize the text box to fit the text content.
    If the auto resize is enabled, the width and height of the text box will be automatically adjusted to fit the text content.
    If the auto resize is disabled, the width and height of the text box will be fixed. You can adjust the width and height of the text box manually.

Tips

The border and fill attributes are also available for all text boxes. The fill can be treated as a text's background. Besides, both the border and fill support gradient.

Shadow attributes

The shadow attributes view has two parts:

  • Shadow switch: Enable or disable the shadow.
  • Shadow button: Click to open the shadow attributes panel as shown in the following screenshot.

There two types of shadow: shadow and inner shadow. The inner shadow is only available for the shape which has a fill. A line, broken line and curve line will not have inner shadow. Becaue they don't have a fill. If you turn off the fill for a shape, the inner shadow will also be disabled for it.

Reflection attributes

The reflection attributes view has two parts:

  • Reflection switch: Enable or disable the reflection.
  • Reflection button: Click to open the reflection attributes panel as shown in the following screenshot.
