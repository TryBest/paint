Create a new image

You can create new images directly in this app. There are several ways to do this.

  • If you have copied or cut some image from another place or app, you can choose the File > New from Clipboard menu command in the top menu bar of this app to create a new image from the clipboard.

  • You can also choose the File > New menu in the top menu bar to create a new empty image.
    After choosing this command to create a new image, a dialog will show up. In this dialog, you can choose the image size, resolution and the background color.

    • Image size. There are many preset image sizes. You can choose one of them or input a custom image size. You can also choose the unit of the image size. The unit can be pixels, inches, cm, mm and feet.

      • The From Clipboard preset size will use the size of the image in the clipboard.
      • The Full Screen preset size will use the size of your current Mac's screen.
      • The Full Screen View Port preset size is similar to the Full Screen preset size, but it will not includig the menu bar and the dock.

    • Resolution. The resolution of your current Mac's screen will be used by default. You are advised to use the default value if you don't have a specific resolution to use.

    • Background color. This is the background color of the new image. You can choose the clear color if you don't want to have a background.

    Tips

    All the setting in this dialog can be modified freely after your image is created. So don't worry about using a wrong image size, resolution and background color in this dialog at this time.

    To modify the image size and resolution, you can choose the Image > Adjust Size... menu in the top menu bar.

    To modify the background color, you can click the image background color button in the bottom bar).

  • You can also create a new image in the open panel.

After creating a new image, a window will show up automatically and you can draw, sketch, or edit your image at there.
