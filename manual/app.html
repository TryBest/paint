Open app

After installing the app from App Store, you can open it by clicking the app icon in the launchpad or in the dock if it is pinned there.

Tips

You can also drag an image or file onto the app icon to open it.

Tips

After opening this app, you can right-click its icon in the dock to show the context menu. Then you can choose the Options > Keep in Dock menu to pin the app in the dock.

When the app is opened, it will show the top menu bar and the open panel.

Top menu bar

The top menu bar has the following menus as shown in the screenshot above.

App menu

The app menu has some useful commands.

  • About. It shows the version of the app.

  • Settings. It shows the settings panel of this app. To get more details about the app settings, please refer to App settings.

  • Upgrade to Full-featured Version. It shows a panel and you can upgrade to the full-featured version of this app or check your current license status in this panel.

  • Quit. It quits the app.

File menu

This menu has some important commands. With them you can:

  • Open an existing image. To know more details about this, please refer to Open existing file.

  • Open a recent image. To know more details about this, please refer to Open existing file.

  • Create a new image. To know more details about this, please refer to Create new image.

  • Create a new image from the clipboard. To know more details about this, please refer to Create new image.

  • Save and export your image. To know more details about this, please refer to Save and export image.

  • Revert your image to some old version.
    If you have made some changes to your image and don't want to have them again, you can revert to some old version which you have saved manually by choosing the File > Save menu command.

  • Duplicate your image.

  • Close your image.

  • Print your image. To know more details about this, please refer to Print image.

Edit menu

This menu has some commands, such as cut, copy, paste, undo, redo, select all, select all objects etc. You should notice the following commands:

  • Select All. This command will select the whole part of your image.

  • Select All Objects. This command will select all objects on the image.

  • Paste Drawing Style. If you have copied/cut some object, This command will apply the drawing style of the copied object to the selected objects. The drawing style includes the line color, line width, fill color, fill gradient, shadow and so on.

Tips

If you select some objects on the canvas, the Cut, Copy, Paste and Delete commands will take effect on the selected objects. However, if there is no selected objects and you use a selelct tool to select some part of the image, these commands will take effect on the selected image part.

Tool menu

This menu helps you to select a tool to use and most commands in it support keyboard shortcuts. To know more about all tools, please refer to Use tools.

Image menu

This menu has commands to edit the whole image, not the selected objects and image part. It has the following commands:

  • Crop. It has the same function as the Crop tool. To know more about crop, please refer to Crop tool.

  • Adjust Size. It will show the resize panel in which you can input a new size and resolution and apply it to the image. To know more about resizing, please refer to Resize image.

  • Adjust Colors. It will show the adjust colors panel in which you can adjust the colors of the image. To know more about color adjustment, please refer to Adjust image colors.

  • Invert Colors. It will invert the colors of the image.

  • Adjust Image Opacity. It will show the opacity panel in which you can adjust the opacity of the image.

  • Remove Background. It will remove the background of the image. To know more about removing background, please refer to Remove background.

  • Rotate Canvas Left. It will rotate the whole image 90 degrees to the left.

  • Rotate Canvas Right. It will rotate the whole image 90 degrees to the right.

  • Flip Canvas Horizontally. It will flip the whole image horizontally.

  • Flip Canvas Vertically. It will flip the whole image vertically.

Object/Layer menu

Unlike the image menu, commands in this menu target the selected objects, no the entire image. In another words, you can use the commands in this menu to manage and edit the selected objects on the canvas. So the commands in this menu are available and enabled until you select some objects on the canvas, or these commands are disabled.
To get more details about these commands, please refer to Manage and edit layers/objects.

View menu

This menu has some commands to config the image canvas view and layers view. It has the following commands:

  • Zoom In. It will zoom in the canvas.
  • Zoom Out. It will zoom out the canvas.
  • Actual Size. It will zoom the canvas to the actual size of the image.
  • Zoom to Fit. It will zoom the image to fit the whole canvas area.
  • Ruler Unit. It can change the unit of the canvas rulers.
  • Ruler Start Point. It can change the start point of the canvas rulers.
  • Show Sidebar. It will show/hide the layers view which is on the left of the canvas view.
  • Show Canvas Grid. It will show/hide the grid on the canvas.

    Tips

    If the grid distance is too short, the grid may not show even if you have choosed to show the canvas grid. If so, please zoom the canvas in to show the canvas grid.

  • Show Canvas Rulers. It will show/hide the rulers on the canvas.
  • Config Canvas Grid. it will open the app setttings panel so that you can configure the grid of the canvas, including the grid color and grid distance.

Window menu

This menu has some commands to manage the windows of the app.

Help menu

This menu has some commands to help you to get help about the app. It has the following commands:

  • Export Favorite Colors. It will export your favorite colors to a file so that you can sync your favorite colors to other devices.

  • Import Favorite Colors. It will import your favorite colors from a file so that you can sync your favorite colors from other devices.

  • Contact Us. It will tell us our support email address.

  • Home page. It will open the home page of the app.

  • Online Help. It will open the online user manual page of the app.

  • Frenquently Asked Questions. It will open the FAQ page of the app.

  • Feedback Issues. When choosing this command, you will be asked to select a file to save your app logs. Then you can send this file to us by email support@trybestapps.com to help us to solve your problem.

Open panel

If no image is opened, the open panel will also be showed when the app is opened.

  • Open recent image.
    The open panel lists all recent images you have opened, so that you can re-open them easily. To do this, please just select one of the recent files and then click on the "Open" button or press the enter key on your keyboard to open it. You can also double click one recent file to open it.

    Tips

    You can change the number of recent files listed. Please visit the following web page to know how to change it.
    https://support.apple.com/guide/mac-help/open-recently-used-items-on-mac-mchlp2724/mac
    Please don't forget to choose the right version of your macOS in this page.

  • Open other image.
    Not only recent documents, but you can also click the Open Other... button in the open panel to open other images from the file system.

  • Create new image.
    If you want to create a new image, please click the Create New button.

  • Create new image from the clipboard.
    If you have copied or cut some image from another place or app, you can click the New from Clipboard button to create a new image from the clipboard.
