Choose a gradient

Gradients are used in some cases, such as the following:

  • Choose a gradient for the line/border of some object.
  • Choose a gradient for the background/fill of some object.
  • Choose a gradient for some texts.

So, you need to choose a gradient sometimes.

Gradient picker panel

You can click the corresponding gradient picker button in all cased above to pop up the gradient picker panel as shown in the following screenshot:

The gradient picker panel can be divided into the following parts:

  • Gradient type segment button.
    It has two options: linear and radial. You can click this segment to toggle between them.

  • Gradient preview and colors.
    All colors of the current gradient are displayed here. You can add new colors or edit and remove existing ones.

  • Invert gradient button.
    You can click this button to invert the current gradient.

  • Gradient list.
    All built-in gradients and your favorite gradients are displayed here.

Modify current gradient

To modify the current gradient, you can:

  • Click the gradient type segment button to toggle between linear and radial.

  • Click the gradient preview area to add a new color to the gradient.

  • Click an existing color to pop up the color pick panel. Then you can choose a color to use it.

  • Drag an existing color to adjust its position.

  • Click an existing color and then press the delete key on your keyboard to delete it.

  • Click the Invert button to invert the current gradient.

  • You can also adjust the center of a radial gradient and the degree of a linear gradient by dragging the corresponding control knobs on the image canvas.

Add current gradient into the favorites

If you want to save the current gradient to use it later, you can click the + button on the bottom of the gradient picker panel. Then the current gradient will be added to your favorite gradients.

Use an existing gradient

All built-in gradients and your favorite gradients are displayed at the bottom of the gradient picker panel. You can click one to use it.
