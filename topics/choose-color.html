Choose a color

Colors are widely used in the app, such as the following cases:

  • Choose a color for the line or border of some object.
  • Choose a color for the background or fill of some object.
  • Choose a color for some texts.
  • Choose a color for a shadow.
  • Choose a color for some gradient.
  • Choose a color for an outline.
  • Choose a color for the canvas grid.

So, you often need to choose a color and use it.

Choose a color from the color picker panel

Regularly, you can click the corresponding color picker button in all cased above to pop up the color picker panel as shown in the following screenshot:

You can choose a color from the picker panel in 3 modes.

1. Common colors mode
Some common colors will be listed here. You can click the one you need to use it.

2. All colors mode
If you want to use a color not listed in the common colors. You can switch to the all colors mode.

Tips

In this mode, you can input and adjust the opacity of the current color.

3. Favorite colors mode
The colors that you have added into the favorites will be listed here.

  • You can click the add button at the right-top of the color picker panel to add the current color into the favorites list.
  • You can click the edit button at the end of each color to edit its name.
  • You can click the delete button at the end of each color to remove it from the favorites list.
  • You can drag and drop some colors to reorder them.

Tips

If you want migrate your favorite colors to another Mac, you can choose the Help > Export Favorite Colors... menu in the top menu bar to export the favorite colors as a file. Then you can import it on another Mac with the Help > Import Favorite Colors... menu.

Pin the color picker panel on the screen

Regularly, the color picker panel will pop up when you click some color picker button and hide when you don't need it. But sometimes, you don't want this. Instead, you want to pin the color picker panel on the screen. There are 3 ways to do this.

  1. Double-click the color picker button to open the color picker panel.

  2. Drag the poped-up color picker panel to a desired position you want.

  3. Click the pin button at the left-top of the color picker panel.

Input a color manually

Sometimes, you know exactly a color and want to use it directly. Then you should input the color manually. Please click the Input Color Manually button at the bottom of the color picker panel. And a color input panel will be shown as the following screenshot:

You can choose to input your color in decimal or hexadecimal format.

Pick a color from the canvas

Sometimes, you want to pick a color from the canvas and use it. Then you can use the eyedropper tool. You can refer to eyedropper tool to learn how to use it.

After picking a color with this tool, a panel will be poped up. And you can click the Add to Favorites button to add the color into the favorite list. Then you can choose the color in all color picker panels by switching to the favorite colors mode.
