Curved texts

Add curved texts

There are two ways to add curved texts.

  • Add curved texts directly on the canvas.

    1. Please choose the Object > Add Curved Texts... menu in the top menu bar.
    2. In the dialog that appears, you can choose a path for the curved texts and then click the OK button.
    3. Type the texts you want to add.
    4. Click the esc key on the keyboard to finish typing.
    5. Adjust the path and shape of the curved texts as you want if needed.

  • Add curved texts on to existing shape or path.

    1. Please select an existing shape or path on the canvas that you want to add curved texts to.

    Tips

    If there isn't such a shape and path or you don't want to use an exiting shape and path, please choose the curve line tool, broken line tool, ellipse tool or vector pen tool in the tools bar and use it to add a new shape or path.
    You can delete the new added shape or path if you want after adding some curved texts.

    1. Please choose the Object > Add Curved Texts To Current Object... menu in the top menu bar.
    2. Type the texts you want to add.
    3. Click the esc key on the keyboard to finish typing.
    4. Adjust the path or shape of the curved texts as you want if needed.

Adjust the shape of curved texts

When curved texts are selected, you can adjust its shape by dragging their control knobs on the canvas.

Edit texts of a curved text box

To edit texts of a curved text box, you can select it using the select and move tool and then double-click on it or press the enter key on your keyboard to start editing texts.

To end editing texts, you can press the esc key on your keyboard or right-click on the canvas to show the context menu and then choose the End Typing menu.
